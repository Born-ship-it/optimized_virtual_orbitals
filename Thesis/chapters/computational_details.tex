The choice of basis set determines the size of the virtual orbital space and hence the difficulty and accuracy of the OVOS calculation. This section describes the basis sets employed and provides the resulting orbital space dimensions for each molecule. See Basis Set Exchange \cite{pritchard2019a} for the source of all basis sets used in this work.

\textbf{6--31G:} A split-valence double-zeta basis set developed by Pople et. al. \cite{ditchfield1971a}\cite{dill1975a}\cite{hehre1972a}. It consists of a minimal basis for core electrons and a double-zeta basis for valence electrons, allowing for improved flexibility in describing electron correlation compared to minimal basis sets.

\textbf{cc--pVDZ:} The correlation-consistent polarized valence double-zeta basis set developed by Dunning \cite{dunning1989}. It includes polarization functions that allow for a better description of electron correlation effects, particularly important for accurate energy calculations.

For all calculations, the molecule is initialized using PySCF with \texttt{gto.Mole()} and the Hartree-Fock reference is computed as \texttt{mf} (see Section~\ref{sec:ref_scf}). 

The orbital optimization parameters are specified as follows: \texttt{num\_opt\_virtual\_orbs} specifies the number of active virtual spin-orbitals to optimize, \texttt{mo\_coeff} provides the orbital coefficients, and \texttt{init\_orbs} selects the initial orbital type (RHF or UHF).

Convergence is monitored via two criteria: a change in correlation energy of $|\Delta E_{\rm corr}| < 10^{-8}$\,Hartree and a gradient norm of $|\Delta G_{\rm corr}| < 10^{-6}$\,Hartree. Optimization proceeds for a maximum of 1000 iterations. Once convergence is achieved, consistent convergence is verified over \texttt{keep\_track\_max}~$= 50$ consecutive iterations to ensure stability.
