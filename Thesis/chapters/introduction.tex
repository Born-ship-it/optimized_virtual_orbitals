\section{The Virtual Orbital Space Problem}
% ---------------------------------------------------------------
% [SECTION INTRO – 1.2]
%   "The virtual orbital space — the set of unoccupied molecular orbitals
%    that describe excited configurations in post-HF methods — grows
%    rapidly with basis set size and dominates the computational cost of
%    correlated calculations. This section surveys strategies for
%    reducing or optimising the virtual space, motivating the OVOS approach."
%
% [WHAT TO COVER]
%   - Standard virtual orbital set: canonical HF virtuals.
%     Problem: not ordered by contribution to correlation energy.
%   - Natural orbitals (L\"owdin 1955 \cite{Lowdin1955}):
%     diagonalise 1-RDM → ordered by occupation number.
%     But require a correlated calculation first — circular dependency.
%   - Frozen natural orbitals (FNO): compute MP2 NO from cheap calculation,
%     truncate by occupation threshold. Widely used in practice.
%   - Pair natural orbitals (PNO): local pair-specific truncation
%     (DLPNO-CCSD(T) can handle hundreds of atoms).
%   - OVOS \cite{Adamowicz1987}: the METHOD of this thesis.
%     Systematically optimise a subset of $N' < N_{\rm virt}$ virtual orbitals
%     to maximise the second-order correlation energy.
%     No pre-existing correlated density needed.
%
% [LEAD-IN TO §1.3]
%   "Beyond classical computing, the problem of a large virtual orbital
%    space has direct implications for quantum algorithms, where qubit
%    counts are the primary constraint. The following section describes
%    this quantum computing context."
% ---------------------------------------------------------------

\section{Quantum Computing Context}
% ---------------------------------------------------------------
% [SECTION INTRO – 1.3]
%   "Quantum computing offers the prospect of efficient simulation of
%    quantum systems via algorithms such as the Variational Quantum
%    Eigensolver (VQE) and Quantum Phase Estimation (QPE). However,
%    current NISQ devices are severely qubit-limited, making the
%    connection between orbital-space reduction and quantum resource
%    requirements directly relevant to this thesis."
%
% [WHAT TO COVER]
%   - Qubit mapping: Jordan-Wigner or Bravyi-Kitaev\cite{Seeley2012};
%     $Q = 2N_{\rm orb}$ qubits for $N_{\rm orb}$ spatial orbitals.
%   - VQE \cite{Peruzzo2014}: variational state preparation + classical optimiser.
%     UCCSD ansatz: single and double excitations from reference state.
%     Circuit depth $\propto N_{\rm occ}^2 N_{\rm virt}^2$.
%   - NISQ constraints: typical current hardware 50–127 qubits,
%     shallow circuits only (few hundred two-qubit gates).
%   - OVOS connection:
%     Reducing $N_{\rm virt} \to N'_{\rm virt}$ reduces $Q$ by $2(N_{\rm virt}-N')$
%     and circuit depth proportionally.
%   - Brueckner connection: OVOS MOs approximate Brueckner orbitals →
%     T1 amplitudes suppressed → UCCSD reduces towards UCCD (shallower).
%   - Cite \cite{Cao2019, deGracia2024, Seeley2012}.
%
% [LEAD-IN TO §1.4]
%   "From this landscape of motivation, the concrete research
%    objectives of this thesis are stated in the following section."
