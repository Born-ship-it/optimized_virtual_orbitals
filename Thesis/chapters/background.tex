% WRITING TIPS – BRUECKNER ORBITALS
% ---------------------------------------------------------------
% What are Brueckner orbitals?
%   - Defined by Brueckner (1954) \cite{Brueckner1954}: orbitals in which
%     the HF single-particle Green's function diagonal, or equivalently
%     where the T1 coupled-cluster amplitudes t_i^a = 0 for all i,a.
%   - Brueckner orbitals maximise the overlap of the HF determinant
%     |Φ_0⟩ with the exact (FCI) ground state |Ψ_0⟩.
%   - In practice computed via the Brueckner-CCD (B-CCD) method
%     \cite{Handy1989}: iterate CC doubles amplitudes while rotating
%     orbitals to eliminate T1.
%   - Why are they better than HF orbitals?
%     The HF reference has non-zero T1 amplitudes when the density
%     matrix from correlation differs from the HF density — this wastes
%     computational effort on single excitations. Brueckner orbitals
%     minimise this waste.
%   - Connection to orbital-optimised MP2 (OO-MP2) \cite{Lee2018}:
%     OO-MP2 variationally minimises the MP2 energy w.r.t. all orbital
%     rotations. At convergence the gradient of the MP2 energy
%     vanishes, which is the same stationarity condition as for
%     Brueckner orbitals at the MP2 level. OO-MP2 orbitals are
%     therefore the MP2-level Brueckner orbitals.
%   - Connection to OVOS \cite{Adamowicz1987}: OVOS minimises the
%     MP2 energy in a restricted active virtual space — conceptually
%     the same orbital stationarity condition, but applied only to
%     the active block. The OVOS optimised MOs are hence expected to
%     approximate Brueckner orbitals within the truncated space.
%     This is a key novel claim of this thesis; see Theory §3.9.
% ---------------------------------------------------------------
